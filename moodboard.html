06 · Tokens en código src/theme.css

/* ═══ default · SILKROAD (Georgia + Verdana) ═══ */
:root {
  /* Palette cold-tech */
  --cold-white: #F4F4F4;
  --black:      #0D0D0D;
  --grey:       #8A8D91;
  --grey-light: #E9E9E9;
  --blue:       #2F5FDB;
  --red:        #D42A2A;

  /* Fuentes semánticas · Silk Road */
  --f-title:   Georgia, 'Times New Roman', serif;
  --f-display: Verdana, Arial, sans-serif;
  --f-body:    Verdana, Arial, sans-serif;
  --f-ui:      Verdana, Arial, sans-serif;
  --f-link:    Verdana, Arial, sans-serif;
  --f-num:     Georgia, 'Times New Roman', serif;   /* italic */
  --f-serif:   'Cormorant Garamond', Georgia, serif;
}

/* ═══ alternativos (5 temas cambiables) ═══ */
[data-theme="mssans"]   { --f-body: 'MS Sans Serif', Tahoma; /* ... */ }
[data-theme="dos"]      { --f-body: 'Perfect DOS VGA 437'; /* ... */ }
[data-theme="w95fa"]    { --f-body: 'W95FA'; /* ... */ }
[data-theme="terminal"] { --black: #b8d4b0; --cold-white: #0a0f0a; /* ... */ }

Editar UNA variable en theme.css propaga el cambio por todo el sistema. Cambiar de fuente en toda la plataforma = editar una línea.

07 · Reglas de color disciplina Silk Road

  • Fondo siempre cold white #F4F4F4 · nunca cálido/crema
  • Texto negro #0D0D0D jerarquía alta · gris #8A8D91 secundario
  • Azul #2F5FDB sólo en links y estados de navegación · nunca decorativo
  • Rojo #D42A2A reservado para badges cortos: "live", "last chance", "CONEXIÓN RECHAZADA"
  • Georgia italic reservado para logo, subtítulos, y números · nunca para cuerpo largo
  • Verdana para todo lo demás · límpio, sin adorno, sin peso extra salvo botones
